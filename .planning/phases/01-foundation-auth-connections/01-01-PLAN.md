   - Import NextAuth, Credentials, PrismaAdapter, prisma, bcrypt, authConfig
   - Spread `...authConfig`, add `adapter: PrismaAdapter(prisma)`, `session: { strategy: "jwt", maxAge: 24 * 60 * 60 }` (D-20: 24h)
   - Credentials provider `authorize` function:
     a. Find user by email: `prisma.user.findUnique({ where: { email } })`
     b. If !user or !user.password → return null
     c. If !user.isActive → return null (D-05)
     d. If user.lockedUntil && user.lockedUntil > new Date() → return null (D-07)
     e. Compare password with bcrypt.compare()
     f. On mismatch: increment failedLoginAttempts, if attempts >= 4 set lockedUntil = now + 15 min (D-07), return null
     g. On match: reset failedLoginAttempts to 0, lockedUntil to null, set lastLoginAt to new Date(), return user
   - Export `{ handlers, auth, signIn, signOut }`
