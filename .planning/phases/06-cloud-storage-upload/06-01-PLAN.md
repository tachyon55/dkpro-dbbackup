Create `src/lib/s3-upload.ts`. This file is the sole S3 upload abstraction used by the scheduler.

**Imports:**
```typescript
import { S3Client, HeadBucketCommand } from "@aws-sdk/client-s3"
import { Upload } from "@aws-sdk/lib-storage"
import { createReadStream, statSync } from "fs"
import { prisma } from "@/lib/prisma"
import { decrypt } from "@/lib/crypto"
```

**S3 client factory — `createS3Client(settings)`:**
Accepts a `CloudStorageSettings` object. Constructs `S3Client` with:
- `region: settings.region ?? "us-east-1"`
- `credentials: { accessKeyId: settings.accessKeyId!, secretAccessKey: decrypt(settings.secretAccessKey!) }`
- `endpoint: settings.endpoint ?? undefined` — only pass if non-null (S3-compatible services like NCP, MinIO)
- `forcePathStyle: !!settings.endpoint` — required for non-AWS S3-compatible endpoints

**Main export — `uploadToS3(filePath: string, connectionName: string, fileName: string): Promise<{ key: string, bucket: string, size: number }>`:**
1. Load settings: `const settings = await prisma.cloudStorageSettings.findFirst()`
2. If no settings or missing required fields (bucket, accessKeyId, secretAccessKey), throw `new Error("클라우드 스토리지 설정이 없습니다")`
3. Build S3 key: `${connectionName}/${new Date().toISOString().slice(0, 10)}/${fileName}`
   - Use `encodeURIComponent` on each path segment to handle spaces/special chars
4. Get file size: `const { size } = statSync(filePath)`
5. Create read stream: `const stream = createReadStream(filePath)`
6. Create `Upload` instance from `@aws-sdk/lib-storage`:
   ```typescript
   const upload = new Upload({
     client: createS3Client(settings),
     params: {
       Bucket: settings.bucket!,
       Key: key,
       Body: stream,
     },
     queueSize: 4,       // parallel upload parts
     partSize: 5 * 1024 * 1024,  // 5MB minimum per AWS spec
     leavePartsOnError: false,
   })
   ```
7. `await upload.done()`
8. Return `{ key, bucket: settings.bucket!, size }`

**Test connection export — `testS3Connection(): Promise<{ success: boolean, message: string }>`:**
1. Load settings via `prisma.cloudStorageSettings.findFirst()`
2. If missing, return `{ success: false, message: "설정이 없습니다" }`
3. Create S3 client, call `HeadBucketCommand({ Bucket: settings.bucket! })`
4. Try/catch: success → `{ success: true, message: "연결 성공" }`, error → `{ success: false, message: err.message }`
