# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-04-01

**Total Plans:** 18

## Overview

기존 Delphi 데스크톱 프로그램(DataBase Backup Manager)을 Next.js 기반 Web App으로 마이그레이션한 초기 완전 기능 버전. 인증/권한 관리, 6종 DB 연결 관리, 수동/스케줄 백업, 실시간 진행 표시, 히스토리 조회, SQL 쿼리 실행기, 클라우드 스토리지 업로드를 모두 포함.

## Milestone Summary

**Key Decisions:**
- Next.js 15 App Router + TypeScript + PostgreSQL + Prisma + NextAuth.js v5 스택 확정
- AES-256-GCM 암호화 키는 env var 전용 — DB에 절대 저장하지 않음
- 백업 실행: child_process.spawn() + 배열 args — exec() 문자열 인터폴레이션 셸 인젝션 방지
- WebSocket: Socket.io on custom server.ts — App Router에 네이티브 WebSocket 없음
- Scheduling: node-cron v4 (단일 서버; 멀티 인스턴스 필요 시 BullMQ 전환 경로 확보)
- Oracle 드라이버: any 타입 + eslint-disable — @types/oracledb 의존성 없이 처리
- Slack 알림: plain fetch POST (Incoming Webhook) — @slack/web-api SDK 오버헤드 불필요
- S3 업로드: @aws-sdk/lib-storage Upload — putObject 대신 멀티파트 자동 처리
- S3 secretAccessKey: GET 응답에서 __masked__ 반환 — 복호화 값 클라이언트 노출 금지

**Issues Resolved:**
- Oracle expdp / SQL Server stdout 비스트리밍 문제 → 드라이버별 isStdoutDump() 분기 처리
- node-cron v4 API 변경 (TaskOptions.scheduled 제거 → 생성 즉시 시작) — 버전별 분기 확인
- Zod 4 .issues vs .errors API 변경 — 모든 validation 라우트 수정
- BigInt fileSizeBytes Next.js 직렬화 오류 → string 변환 후 클라이언트 전달
- shadcn CLI npm --legacy-peer-deps 충돌 → Radix 프리미티브 수동 설치 + 컴포넌트 직접 작성
- Monaco 에디터 SSR 오류 → dynamic import with ssr:false

**Technical Debt Incurred:**
- Phase 2 연구 플래그: Oracle expdp / SQL Server BACKUP DATABASE TO DISK 진행률 전략 미완 (stdout 스트리밍 없음)
- Phase 3 연구 플래그: 서버 재시작 시 중단 스케줄 복구 구현 확인 필요
